# Sales Analytics Dashboard

## 📌 Overview

The **Sales Analytics Dashboard** is a Python-based data analysis project that transforms raw sales data into meaningful business insights. It performs data cleaning, calculates key sales metrics, generates visualizations, and exports reports to help understand sales performance.

This project demonstrates practical data analysis skills using Python and is designed to showcase knowledge of data manipulation, reporting, and visualization.

---

## ✨ Features

* Load sales data from CSV files
* Clean and preprocess raw datasets
* Remove duplicate records
* Handle missing values
* Calculate important sales KPIs
* Analyze product and category performance
* Generate insightful visualizations
* Export cleaned datasets
* Generate a detailed analytics report

---

## 🛠️ Tech Stack

* Python
* Pandas
* Matplotlib

## 📊 Analysis Performed

The project calculates and analyzes:

* Total Orders
* Total Revenue
* Average Order Value
* Highest Order Value
* Lowest Order Value
* Total Quantity Sold
* Best Selling Products
* Category-wise Revenue
* City-wise Sales
* Payment Method Distribution
* Monthly Sales Performance

## 📈 Visualizations

The dashboard generates the following charts:

* Monthly Sales Trend
* Revenue Distribution
* Top Selling Products
* Sales by Category
* Sales by City
* Payment Method Distribution

All charts are automatically saved inside the `output/charts/` directory.
